# Crónicas de un Acoso Exitoso

Web tipo libro interactivo hecha con Next.js, TypeScript, Tailwind CSS y Framer Motion.

## Requisitos

- Node.js 20 o superior
- npm 10 o superior

## Ejecutar en local

1. Instala dependencias:

```bash
npm install
```

2. Inicia el servidor de desarrollo:

```bash
npm run dev
```

3. Abre:

```txt
http://localhost:3000
```

## Imagen de spoiler final

- Coloca tu imagen real en:

```txt
/public/images/spoiler-final.jpg
```

- Si no existe, la última página mostrará automáticamente un placeholder.

## Scripts

- `npm run dev`: desarrollo
- `npm run build`: build de producción
- `npm run start`: ejecutar build
